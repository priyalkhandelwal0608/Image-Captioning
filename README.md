#  Image Caption Generator

An **Image Captioning Web App** built using **Gradio** and the **BLIP model** (`Salesforce/blip-image-captioning-base`). Upload an image and get a descriptive caption automatically generated by the model.

---

## Features

- Upload images in any standard format (PNG, JPG, etc.).
- Generates human-like captions using a **state-of-the-art BLIP model**.
- Adjustable parameters:
  - **Max Caption Length**: Control the length of the generated caption.
  - **Beam Search Size**: Trade-off between diversity and accuracy.
  - **Temperature**: Adjust randomness (higher → more creative captions).
- Interactive **Gradio UI** for easy experimentation.

---



## Installation and run
pip install -r requirements.txt
python app/main.py
